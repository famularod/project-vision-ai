# Project Vision AI Roadmap

## Current Phase

UX Simplification and Value Alignment

The current phase focuses on making the app faster, clearer, and more valuable for daily project-management workflows. The priority is not adding more surface area. The priority is making existing capabilities easier to find, easier to understand, and useful within 60 seconds.

## Completed Foundation

The app already has important technical and product foundations:

- OpenAI integration
- Supabase integration
- Local/cloud sync foundation
- Project creation and project management
- Project update capture
- Photo-based field updates
- Admin/More area
- Diagnostics and cloud status tooling
- Reports foundation
- Schedule import and schedule summary foundation

## Immediate Priorities

### 1. Simplify Home

Goal:

- Make Home answer "what should I do next?"

Focus:

- Current or last active project
- Capture Update as the primary action
- Small project attention section
- Small schedule attention section
- Recent activity
- No admin/debug clutter

### 2. Build True Reports Hub

Goal:

- Make Reports a clear destination for communication.

Focus:

- Executive Report
- Weekly Executive Report
- Project Health Report
- Photo Log Report
- Open Issues Report
- Critical Path Report
- Milestone Report
- Saved Updates / History

### 3. Make Schedule Uploads Visibly Useful

Goal:

- Ensure uploaded schedules immediately produce visible value.

Focus:

- Import result summary
- Upcoming tasks
- Overdue tasks
- Milestones
- Critical/high-risk items
- Tasks by project
- Tasks by area
- Missing mapping review
- Schedule signals on Home, Projects, Reports, and AI Coach

### 4. Add Talk-to-Text

Goal:

- Let project managers capture field context quickly without typing.

Focus:

- Dictate update notes
- Dictate photo captions
- Dictate action items
- Clean up spoken text into project-ready language
- Preserve user review before saving or sending

### 5. Add Project Brief

Goal:

- Provide a fast, useful one-page project status view.

Focus:

- Project summary
- Current status
- Latest update
- Open actions
- Overdue work
- Schedule health
- Upcoming milestones
- Risks
- Recommended next action

### 6. Add Audience-Specific Updates

Goal:

- Help users communicate the same project information differently depending on audience.

Focus:

- Boss/executive update
- Customer update
- Contractor follow-up
- Internal PM notes
- Safety-focused update
- Issue escalation update

## Guiding Rule

Every roadmap item must support the product mission:

Project Vision AI exists to reduce the time it takes a project manager to understand, communicate, and make decisions about a project.

Every roadmap item must also support the 60-second workflow rule.
